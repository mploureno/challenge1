# Challenge#1

This is a web app done to meet a little RoR challenge set by a friend... a pro friend :sunglasses:

#### Requirements

- Ruby 2.6.3
- Rails 5.2.3
- PostgreSQL 9.5+

#### Setup

To get all running do the following:
 ```shell
 bundle
 ```

Setup `config/database.yml` then run:
```shell
 rails db:create 
 rake db:migrate
```

Test your configuration:
```shell
 rails s 
```
